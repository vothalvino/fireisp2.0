## User Experience Flow

```
First Time User:
1. Access application URL
2. Redirected to Setup Wizard
3. Create root user account
4. Configure SSL (or skip)
5. Enter company information
6. Redirected to Dashboard
7. Start adding clients and services

Regular User:
1. Login with credentials
2. View Dashboard statistics
3. Navigate to desired section
4. Perform actions (CRUD operations)
5. View real-time updates
6. Logout when done

Admin Tasks:
- Add clients
- Assign services
- Configure Mikrotik routers
- Monitor sessions
- View bandwidth usage
- Manage billing
```
